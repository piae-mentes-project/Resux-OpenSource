# 目录简述
存放实用的代码，目前有：
 - `Data` 实用的数据结构，有利于更好的处理数据
 - `Extension`扩展方法，方便功能的编写
 - `Helper` 实用的助手类，有利于其他代码的编写，减少代码的重复
 - `ResourceLoader`资源加载



## Data

常用的一些数据结构

## Extension

类型的方法扩展

- ArrayExtension：数组类型的方法扩展
- EnumExtension：枚举类型的方法扩展
- StringExtension：字符串类型的方法扩展

## Helper

一些常用工具

## ResourceLoader

资源加载器相关代码

包括Resources加载和AssetBundle加载，且都拥有**泛型类**

# 目录简述
用于存放“游戏游玩”相关的代码，如Note的Script/各种数据结构/判定相关代码



> GamePlayer.cs  -  音游玩法主要代码
>
> MusicPlayer.cs  -  音乐播放代码
>
> MusicScoreRecorder.cs  -  成绩记录器



**目录：**

- /Note：半note和判定note相关
- /Data：游玩时的相关数据
- /Effect：游玩时的特效
- /Judge：判定方法
